## To use you should do

Clone the project repository

```bash
git clone https://github.com/TrueGelen/qraphql_react_test.git
```

Then

```bash
npm i 
```

And then to build project in dev mod u should use (it will be placed on 8080 port)

```bash
npm run dev
```

or build mod

```bash
npm run build
```
